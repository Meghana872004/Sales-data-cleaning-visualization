# Sales-data-cleaning-visualization
End-to-end sales data cleaning and visualization project using Python, Pandas, Seaborn, and Matplotlib.

##  Project Overview
This project demonstrates end-to-end data cleaning and validation using Python.

##  Data Cleaning Steps
- Checked missing values using heatmap
- Filled numeric columns using median
- Filled categorical columns using mode
- Removed invalid age values
- Converted datatypes properly
- Reset index after row removal
- Final validation using visual heatmap and df.info()

##  Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Jupyter Notebook
